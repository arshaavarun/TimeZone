Off-machine copy (cloud folder)

Mirror every backup into another folder — point this at your OneDrive or Google Drive folder to keep a copy off this PC. Only backups are copied there; the live database must stay on this computer — running it from a sync folder corrupts it. Leave blank to turn off.

Restore from a backup

Replace all current data with a backup .db file you saved earlier. Your current data is backed up first, so this is undoable.
